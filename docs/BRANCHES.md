## Cara Verifikasi / Update Koordinat

1. Buka admin panel → **Cabang** → **Edit** cabang yang ingin diperbarui
2. Klik pada map untuk menempatkan marker di lokasi tepat
3. Atau drag marker ke posisi yang benar
4. Klik **Simpan**
