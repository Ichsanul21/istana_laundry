# IstanaLaundry / LaundryOS

Static HTML/JS marketing site — **no build step, no package manager, no backend.**

## Files

- `index.html` — Single-file landing page for Istana Laundry Samarinda
- `TechSheet.md` — Full PRD / TAD / ERD / UI-UX / SRD spec (1381 lines). Read before proposing architecture changes.
- `logo.png` — 225×225 PNG logo

## No build step

Serve with any HTTP server:
```
python3 -m http.server 8000
```

## What's NOT in this repo

No backend, no database, no API, no tests, no CI/CD, no package.json.
